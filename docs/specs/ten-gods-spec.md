# 작업 명세: 십성(十星) + 지장간(支藏干) 계산 추가

> saju-engine 레포. correctPillars 출력에 십성과 지장간을 추가한다. 결제 후 정밀 해석
> (커리어=관성, 금전=재성, 연애=재성/식상 등)의 재료. 기존 로직은 건드리지 않는다.
> 원칙은 CLAUDE.md.

## 목적

결제 후 정밀 해석에 쓸 십성 데이터를 엔진이 제공한다. 천간·지지 대표뿐 아니라 지지 속
지장간에도 십성을 매겨, "겉으로 드러난 것 + 속에 숨은 것"까지 해석할 수 있게 한다.

## 확인된 사실 (실제 사주로 검증)

- 십성 판정 규칙(일간 기준 오행 생극 + 음양)을 검증 명식(일간 경금)으로 검증: 천간·지지
  8개 전부 실제 사주 표와 일치. 규칙 정확함.
- 지장간 표준표(여기·중기·정기)가 검증 표와 일치(자=임계, 인=무병갑, 미=정을기, 신=무임경).
  이 표준 방식으로 확정.
- 천간 음양: 갑·병·무·경·임 = 양(+), 을·정·기·신·계 = 음(-).

## 1. 지장간 테이블 (12지지, 여기·중기·정기 순)

```
자(子): 임·계
축(丑): 계·신·기
인(寅): 무·병·갑
묘(卯): 갑·을
진(辰): 을·계·무
사(巳): 무·경·병
오(午): 병·기·정
미(未): 정·을·기
신(申): 무·임·경
유(酉): 경·신
술(戌): 신·정·무
해(亥): 무·갑·임
```
- 각 지지의 지장간을 배열로. 순서는 여기→중기→정기(정기가 대표, 마지막).
- 지지에 따라 2개 또는 3개.

## 2. 십성 판정 규칙 (일간 기준)

일간과 대상 천간의 (1)오행 생극 + (2)음양 동이로 판정:

| 관계 | 음양 같음 | 음양 다름 |
|------|-----------|-----------|
| 같은 오행 | 비견(比肩) | 겁재(劫財) |
| 일간이 생하는 오행 | 식신(食神) | 상관(傷官) |
| 일간이 극하는 오행 | 편재(偏財) | 정재(正財) |
| 일간을 극하는 오행 | 편관(偏官) | 정관(正官) |
| 일간을 생하는 오행 | 편인(偏印) | 정인(正印) |

- 오행 상생: 목→화→토→금→수→목. 상극: 목극토, 토극수, 수극화, 화극금, 금극목.
- "일간이 생하는" = 일간 오행이 생하는 대상. "일간을 생하는" = 일간을 생해주는 것. (방향 주의)
- 일간 자신(일주 천간)은 십성을 매기지 않음(본인 자리).
- 지지 대표 십성은 지지의 **정기(正氣, 본기) 천간** 음양으로 판정한다. 지지 '위치 음양'을
  쓰면 안 된다. 특히 자(子)·오(午)는 위치상 양이지만 정기(계 癸·음수, 정 丁·음화)가 음이라
  위치 음양과 반대다. 예: 일간 경금 + 일지 자(子) → 정기 계(음수) → 상관(위치 음양으로 하면
  식신이 되어 틀림). 오행은 정기가 곧 지지 본기 오행이라 어느 방식이든 같고, 음양만 갈린다.

## 3. correctPillars 출력에 추가

```ts
tenGods: {
  // 천간 십성 (일간 제외 3개: 연간·월간·시간)
  heavenly: { year: string; month: string; hour: string | null };
  // 지지 십성 (각 지지의 정기 천간 기준 4개: 연지·월지·일지·시지)
  earthly: { year: string; month: string; day: string; hour: string | null };
  // 지장간 십성 (각 지지의 지장간 전부)
  hidden: {
    year: { stem: string; god: string }[];
    month: { stem: string; god: string }[];
    day: { stem: string; god: string }[];
    hour: { stem: string; god: string }[] | null;
  };
};
```
- 시주 관련(hour)은 timeUnknown이면 null(시간 모름 시 시주는 정오 추정이라 제외).
- 지장간 십성은 각 지장간 천간마다 { 천간, 십성 } 쌍으로.

## 4. 구현 (기존 로직 불변)

- 지장간 테이블 상수 추가(src/data 또는 correct-pillars 내부).
- 십성 판정 함수: (일간, 대상천간) → 십성. 오행 생극 + 음양으로.
- correctPillars 끝에서 각 기둥의 천간·지지(정기 기준)·지장간에 십성을 매겨 tenGods 조립.
- 기존 4기둥·연월주·진태양시·오행·일간 로직은 한 줄도 바꾸지 않음. 출력에 tenGods만 추가.
- 12운성·12신살은 이번 범위 아님(추후 해석 풍부화 시 검토).

## 5. 검증

- 기존 테스트 전부 통과(회귀).
- 검증 명식(대구 1992-07-23 05:20, 일간 경금)으로:
  - 천간 십성: 연 식신, 월 정관, 시 편인.
  - 지지 십성: 연 비견, 월 정인, 일 상관, 시 편재.
  - 지장간 십성 예: 월지 미(정·을·기) → 정관·정재·정인.
- 시간 모름 → hour 관련 십성 전부 null.
- typecheck/build, dist 재빌드.
