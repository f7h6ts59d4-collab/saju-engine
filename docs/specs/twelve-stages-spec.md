# 작업 명세: 12운성(十二運星) 계산 추가

> saju-engine 레포. correctPillars 출력의 원국 4기둥과 대운 10사이클 각각에 12운성을
> 얹는다. 일간의 "기세 강약"을 생로병사에 빗댄 지표로, 정밀 해석에서 원국의 타고난
> 기세 + 대운별 기세 흐름을 읽는 재료. 기존 로직 무변경, 리팩토링 구조 준수. 원칙은 CLAUDE.md.

## 목적

12운성은 일간이 각 지지에서 갖는 기운의 강약(장생→목욕→…→양의 12단계)이다.
- 원국 4기둥에 얹으면: 타고난 기세(건록·제왕이면 강, 사·절이면 약).
- **대운 10사이클에 얹으면**(핵심): 10년마다 일간의 기세가 어떻게 흐르는지.
  대운 십성(무슨 기운의 국면)과 겹쳐 "재성 대운인데 일간은 절지라 약한 시기" 같은
  입체적 시간축 해석 가능. 표 하나로 원국·대운 양쪽에 재사용.

## 규칙 (표준 12운성, 5명식 검증 완료)

일간의 **장생지**에서 시작해, 양간은 순행/음간은 역행으로 12지지에 12단계를 배치.
- 12단계 순서: 장생→목욕→관대→건록→제왕→쇠→병→사→묘→절→태→양
- 장생지(화토동법: 무는 병과, 기는 정과 동일):
  갑→해, 을→오, 병→인, 정→유, 무→인, 기→유, 경→사, 신→자, 임→신, 계→묘
- 양간(갑·병·무·경·임): 장생지에서 지지 순행(자→축→인…)으로 단계 진행.
- 음간(을·정·기·신·계): 장생지에서 지지 역행(자→해→술…)으로 단계 진행.
- 계산: 일간 장생지 인덱스 기준, (지지 - 장생지) × 방향 → 12단계 매핑.

> 검증: 5명식(경·기·계·갑·을) 원국 4기둥 20셀 전부 실제 포스텔러 12운성 열과 일치.

## 출력 (correctPillars에 추가)

### 원국 12운성 — 각 기둥에 필드 추가 또는 별도 객체
```ts
twelveStages: {
  year: string; month: string; day: string; hour: string | null;
};
```
- 일지(일간 자신의 자리)도 계산한다(일간 대비 일지). 시주는 timeUnknown이면 null.

### 대운 12운성 — 기존 majorLuck.cycles 각 항목에 추가
```ts
// majorLuck.cycles[i] 에 필드 추가:
{
  startAge, endAge, pillar, pillarHanja, heavenlyGod, earthlyGod,
  lifeStage: string;   // 이 대운 지지의 12운성 (일간 기준)
}
```
- 대운 간지의 **지지**에 일간 기준 12운성을 매긴다.

## 검증

- 기존 테스트 전부 통과(회귀).
- 원국(5명식 대조):
  - 경(임신·정미·경자·무인): 연 신=건록, 월 미=관대, 일 자=사, 시 인=절
  - 기(경오·무인·기유·신미): 연 오=건록, 월 인=사, 일 유=장생, 시 미=관대
  - 계(기사·임신·계축·신유): 연 사=태, 월 신=사, 일 축=관대, 시 유=병
  - 갑(경오·신사·갑술·임신): 연 오=사, 월 사=병, 일 술=양, 시 신=절
  - 을(경오·무인·을사·갑신): 연 오=장생, 월 인=제왕, 일 사=목욕, 시 신=태
- 대운(경 명식, 일간 경): 5세 무신=건록, 15세 기유=제왕, 25세 경술=쇠, 35세 신해=병,
  45세 임자=사, 55세 계축=묘, 65세 갑인=절, 75세 을묘=태
- 시간 모름 → 원국 시주 twelveStages.hour = null (대운은 성별 있으면 그대로).
