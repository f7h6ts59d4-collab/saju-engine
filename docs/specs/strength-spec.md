# strength-spec: 신강/신약 판정 모듈 명세 (v1 초안)

리프 모듈. correctPillars 오케스트레이터에 `strength` 필드로 통합.
기존 출력에 필드 추가만 하며, 기존 소비자를 깨지 않는다.

## 채택 학파 조합 (스위치 고정 — 변경 시 이 문서 개정 필수)

| 스위치 | 고정값 | 근거 |
|---|---|---|
| 득령 기준 | 월지 정기의 십성이 비겁 또는 인성이면 득령 | 자평 정통 다수파. 카논 명식에서 정기법·사령법·왕상론 3방식 일치 확인 |
| 조토생금 | 인정 (미·술토도 금을 생한다) | 조토무생금설 미채택. 조열함은 조후 필드로 분리 표시 |
| 통근 | 지장간에 일간과 같은 오행이 있으면 통근. 해당 지지의 12운성이 건록·제왕이면 가중 | 현대 억부 통설 |
| 합반·충 감점 | 천간합에 묶인 천간, 충 맞은 지지는 세력 50% 감점 | 합거·합반 논리의 보수적 적용. relations 출력의 존재 사실만 사용 |
| 합화(化) | 미반영 | 엔진 전역 원칙(합화 판정 금지)과 동일 |
| 반합·방합·형·파·해·원진 | 강약 판정에 미반영 (v1) | 세력 형성/손상 해석은 해석층 몫 |
| 조후 | 강약 점수에 합산 금지. 별도 필드로만 출력 | 포스텔러 실패 지점(온도의 오행% 환산 합산) 회피 |

## 판정 알고리즘

### 1. 글자 분류
일간을 제외한 7글자 + 전체 지장간을 십성 그룹으로 분류.
- 아군(support): 비겁·인성
- 적군(drain): 식상·재성·관성

### 2. 가중치 (v1 확정값)

| 자리 | 가중치 |
|---|---|
| 월지 (정기) | 30 |
| 일지 (정기) | 15 |
| 연지 (정기) | 10 |
| 시지 (정기) | 10 |
| 월간 | 12 |
| 연간 | 10 |
| 시간 | 10 |
| 지장간 여기·중기 (각) | 3 |

- 통근 보너스: 일간과 같은 오행이 지장간에 있는 지지마다 아군 +8.
  해당 지지 12운성이 건록·제왕이면 +12로 상향.
- 감점: 천간합 성립 시 묶인 두 천간의 세력 50%.
  지지충 성립 시 충 맞은 두 지지의 정기 세력 50% (지장간 점수는 유지).
  감점은 relations 모듈 출력을 입력으로 사용한다 (자체 재계산 금지).
  같은 기둥이 여러 감점 관계에 걸려도 50% 감점은 기둥당 1회만 적용한다.
  통근 보너스는 정기 세력이 아니므로 충 감점의 대상이 아니다.

### 3. 비율과 라벨

ratio = support / (support + drain)

**무근 게이트 (통근이 척추)**: 통근이 0이면 (일간과 같은 오행이 어느 지장간에도
없으면) ratio와 무관하게 라벨 상한은 중화다. 인성이 아무리 많아도 뿌리 없는
일간은 신강 계열이 될 수 없다. 반대 방향 게이트는 두지 않는다
(강한 통근만으로 신약을 면제하지는 않음).

| ratio | 라벨 |
|---|---|
| 0.80 이상 | 태강 |
| 0.68 ~ 0.79 | 신강 |
| 0.56 ~ 0.67 | 중화신강 |
| 0.44 ~ 0.55 | 중화 |
| 0.33 ~ 0.43 | 중화신약 |
| 0.21 ~ 0.32 | 신약 |
| 0.20 이하 | 태약 |

경계값·가중치는 v1 제안값 그대로 카논·극단 케이스를 모두 통과해 보정 없이
확정한다(카논 측정값: 아군 63 / 적군 46 / ratio 0.578 → 중화신강).
라벨 경계는 하한 포함 연속 구간으로 구현한다(ratio ≥ 0.80 태강, ≥ 0.68 신강,
≥ 0.56 중화신강, … 순서로 첫 매치).

### 4. 희기 방향 (favorable)

| 라벨 | favorable |
|---|---|
| 태강·신강·중화신강 | 식상·재성·관성 |
| 중화 | null (방향 단정 안 함) |
| 중화신약·신약·태약 | 인성·비겁 |

단일 용신 지목은 엔진 범위 밖 (해석층 보류 사항).

### 5. 조후 (johu) — 별도 필드

- season: 월지 기준 (인묘진=봄 / 사오미=여름 / 신유술=가을 / 해자축=겨울)
- label: 여름=조열, 겨울=한습, 봄=온, 가을=량
- earthFlags: 원국 지지 중 미·술=조토, 진·축=습토 존재 표시
- 점수 합산 금지. 서술용 사실 라벨만.

## 출력 형태

```ts
strength: {
  score: { support: number; drain: number; ratio: number };
  factors: {
    deukryeong: boolean;        // 득령
    deukji: boolean;            // 득지
    roots: Array<{ branch: string; stage: string; boosted: boolean }>; // 통근 상세
    damages: Array<{ kind: '합반'|'충'; targets: string[] }>;          // 적용된 감점
  };
  label: '태약'|'신약'|'중화신약'|'중화'|'중화신강'|'신강'|'태강';
  favorable: '식상·재성·관성' | '인성·비겁' | null;
  johu: { season: string; label: string; earthFlags: string[] };
}
```

## 검증 케이스

### 카논 (필수 통과)
- 1992-07-23 04:30 대구, 남 (임신 정미 경자 무인, 일간 경금)
- 기댓값: 득령 O(미토 정인) / 득지 X(자수 상관) / 통근: 신(건록, 가중) /
  감점: 정임합(정·임), 인신충(인·신) / **라벨 = 중화신강** / favorable = 식상·재성·관성 /
  johu = 여름·조열, earthFlags [미=조토]
- 근거: 대운 실증 역추정 (기유 정인·겁재·제왕 구간 최악 = 과충전,
  경술에서 설기 처방으로 호전, 무신은 지장간 임수 배출구로 양호)

### 극단 케이스 (구현 시 탐색으로 고정)
엔진 자체로 일시를 탐색해 아래 구조를 만족하는 실제 명식을 찾아 테스트에 고정:
- 태강 후보: 비겁·인성이 7자 중 6 이상 + 건록/제왕 통근 + 무충
- 태약 후보: 일간 무근(통근 0) + 아군 1 이하 + 재·관 다수
- 중화 후보: 아군/적군 균형 + 감점 상쇄
어느 학파로 계산해도 결과가 같은(논쟁 여지 없는) 구조여야 한다.

탐색 결과 고정 (대구 Asia/Seoul, 1950~2005 × 12시진 전수 탐색):
- 태강: 1950-01-23 12:30 → 기축 정축 무오 무오. 무토 일간, 7자 전부 화·토
  (아군 7/7) + 오화 제왕 통근 2 + 합·충 감점 없음. ratio 0.925.
- 태약: 1950-01-10 10:30 → 기축 정축 을사 신사. 을목 무근(통근 0) +
  표면 아군 0 + 재관 다수(재 4·관 1). ratio 0.050.
- 중화: 1950-02-18 18:30 → 경인 무인 갑신 계유. 아군 3(정인·비견 2, 건록
  통근 2) ↔ 적군 4(관살 3·재 1), 무계합·인신충이 양편을 같이 깎아 상쇄.
  ratio 0.502.

### 회귀
- 기존 필드(십성·대운·신살·relations 등) 불변.
- strength는 gender 유무와 무관하게 계산된다 (대운과 독립).

## 명시적 비목표 (v1)
- 용신 지목 없음
- 격국 판정 없음
- 조후 점수화 없음
- 종격(從格) 등 특수격 판정 없음 — 극단 ratio도 일반 라벨로만 표기
