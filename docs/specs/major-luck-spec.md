# 작업 명세: 대운(大運) 계산 추가

> saju-engine 레포. correctPillars 출력에 대운을 추가한다. 결제 후 정밀 해석에
> 시간축(10년 단위 운의 흐름)을 부여하는 핵심 재료. 기존 로직은 건드리지 않는다.
> 원칙은 CLAUDE.md.

## 목적

정적인 성격·구조 해석에 시간축을 더한다. "이 사람이 몇 세부터 어떤 기운의 10년을
지나는가"를 계산해, "35세부터 재성의 대운" 같은 흐름 해석을 가능하게 한다.

## 확인된 사실 (실제 대운으로 검증)

- 방향·대운수·간지 나열 규칙을 검증 명식(임신년 정미월 경자일 무인시, 남성)으로 검증:
  순행, 대운수 5, 간지 나열(무신·기유·경술·신해·임자·계축·갑인·을묘)이 실제 값과 일치.
  (검증 값: 5세 무신, 15세 기유 … 65~84세 갑인·을묘. 포스텔러 만세력과도 대조됨.)

## 대운 계산 규칙 (3단계)

### 1. 방향 (순행 / 역행)
- 연간(年干)의 음양 + 성별로 결정:
  - 양년생 남성 · 음년생 여성 → 순행(順行)
  - 음년생 남성 · 양년생 여성 → 역행(逆行)
- 천간 음양: 갑·병·무·경·임 = 양, 을·정·기·신·계 = 음.
- 검증 명식: 임(양년) + 남성 → 순행.

### 2. 대운수 (첫 대운 시작 나이)
- 순행: 출생 시각부터 **다음 절기 입절 시각**까지의 일수 ÷ 3.
- 역행: 출생 시각부터 **이전 절기 입절 시각**까지의 일수 ÷ 3.
- 3일 = 1년 규칙. 나머지 처리는 표준(반올림 또는 버림 — 검증 값 5로 맞춰 확인).
- 절기 입절 시각은 엔진의 정밀 절기 테이블(solar-terms-precise) 사용.
- 검증 명식: 출생(1992-07-23)~다음 절기(입추)까지 약 15일 ÷ 3 = 5.

### 3. 간지 나열
- 월주(月柱)에서 시작해, 순행이면 60갑자 앞으로 / 역행이면 뒤로 진행.
- 각 대운은 10년. 첫 대운은 대운수 나이부터 시작.
- 보통 8~10개 대운을 나열(최대 100세 전후까지).
- 검증 명식: 월주 정미 → 순행 → 무신(5)·기유(15)·경술(25)·신해(35)·임자(45)·
  계축(55)·갑인(65)·을묘(75)…

### 4. 각 대운에 십성 매기기
- 각 대운 간지의 천간·지지에 일간 기준 십성을 매긴다(기존 십성 규칙 재사용).
- 지지 십성은 정기(본기) 천간 기준(십성 작업과 동일 규칙).
- 예: 갑인 대운(일간 경금) → 천간 갑=편재, 지지 인의 정기 갑=편재 → 재성의 국면.

## correctPillars 출력에 추가

```ts
majorLuck: {
  direction: '순행' | '역행';
  startAge: number;            // 대운수(첫 대운 시작 나이)
  cycles: {
    startAge: number;          // 이 대운 시작 나이
    endAge: number;            // 이 대운 끝 나이
    pillar: string;            // 간지(예: '무신')
    pillarHanja: string;
    heavenlyGod: string;       // 천간 십성
    earthlyGod: string;        // 지지 십성(정기 기준)
  }[];
};
```

## 구현 (기존 로직 불변)

- 방향·대운수·간지 나열·십성 매기기를 correctPillars 끝에서 계산해 majorLuck 조립.
- 대운수 계산에 birthUtc와 절기 입절 시각(기존 절기 테이블) 사용.
- 십성 매기기는 기존 십성 판정 함수 재사용(새로 만들지 말 것).
- 기존 4기둥·연월주·진태양시·오행·일간·십성 로직은 한 줄도 바꾸지 않음. majorLuck만 추가.
- 세운·월운·신살은 이번 범위 아님.

## 검증

- 기존 테스트 전부 통과(회귀).
- 검증 명식(대구 1992-07-23 05:20, 남성)으로:
  - 방향 순행, 대운수 5.
  - 간지: 5세 무신, 15세 기유, 25세 경술, 35세 신해, 45세 임자, 55세 계축,
    65세 갑인, 75세 을묘.
  - 십성 예: 갑인(65세) → 천간 갑 편재, 지지 인 편재(재성 국면).
- typecheck/build, dist 재빌드.
